# Weather Trend Analysis 🌤️

Live weather + historical trends + 7 day forecast 
for major Indian cities.

## 🌐 Live Demo
👉 [Try the app here!](https://weather-trend-analysis-fbcswok3ghmivyqzd85dzq.streamlit.app/)

## 🎯 Features
- 🔴 Live current weather
- 📅 7 day forecast
- 📊 5 year historical analysis
- 🌡️ Temperature trends
- 🌧️ Rainfall patterns
- 🔥 Correlation analysis
- 8 major Indian cities supported

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Open-Meteo API (free, no key needed!)
- Streamlit

## 📊 Cities Supported
Mumbai, Delhi, Pune, Bangalore, Chennai, 
Kolkata, Hyderabad, Ahmedabad
